# Touch and Slice

## 게임 컨셉
```
Touch and Slice는 'Fruit Ninja'를 모작하는 게임입니다. 
사용자는 화면을 터치하여, 지나가는 오브젝트를 자르면 점수를 얻습니다. 
자르지 못하고 지나가는 오브젝트의 개수 혹은 잘라서는 안되는 오브젝트를 
자른 횟수가 제한된 기회만큼 누적되면 게임은 종료되고 현재까지 획득한 점수를 보여줍니다.
```
